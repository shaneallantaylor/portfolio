Megan Thompson is a professional photographer working at Hopeless Records and was in dire need of an updated website. I designed the the site with input from her while incorporating best practices for web development, include mobile first and responsive design. It is a single-page application, utilizing HTML5 History, a custom Django backend, URL routing and the Vimeo API. I also wrote a full technical guide for managing the backend through the custom Django installation so Megan is in full control of the content on her site.

The creation of this gallery was a challenge put forth by my friends at Asana. The task was to build a photo gallery with a fixed-width, multiple-row layout. There were also a few constraints to consider when laying out the images in the gallery: The order of images must be maintained, images in the same row must have the same height, but must also maintain their aspect ratio (+/- 1px) and not be cropped, rows are variable-height, but must be less than the maximum row height, every row except the last must have total width (including spacing between images) equal to the container width, and the last row must have total width less than or equal to the container width.
